1. Введение

1.1. Основная информация
Документ описывает методы и подходы к тестированию, которые будут использоваться
тестировщиками для тестирования тренажера «Диск здоровья».
Объект тестирования — это деятельность, направленная на проверку работоспособности
и безопасности тренажера.

1.2. Цель
Тест-план проекта преследует следующие цели:
- Определить существующую информацию о тренажере,
подлежащую тестированию.
- Описать стратегии тестирования, которые будут использоваться.
- Определить необходимые ресурсы для проведения работ по тестированию.
- Привести результаты тестирования.
Результаты будут отправлены заказчику в виде отчетов.

2. Область тестирования тренажера

«Диск здоровья» - это тренажер, представляющий собой два соединенных между собой и  вращающихся относительно друг друга металлических диска диаметром 285 мм. Максимальный вес пользователя 90 кг.

В объем работ по тестированию тренажера входит тестирование следующих
функций:
2.1. вращение диска по часовой стрелке без нагрузки
2.2. вращение диска против часовой стрелки без нагрузки
2.3. вращение диска по часовой стрелке со средней нагрузкой 45 кг
2.4. вращение диска против часовой стрелки со средней нагрузкой 45 кг
2.5. вращение диска по часовой стрелке с максимальной нагрузкой 90 кг
2.6. вращение диска против часовой стрелки с максимальной нагрузкой 90 кг
2.7. отсутствие шума при вращении диска без нагрузки
2.8. отсутствие шума при вращении диска со средней нагрузкой 45 кг
2.9. отсутствие шума при вращении диска с максимальной нагрузкой 90 кг
2.10. гладкость поверхности нижней платформы диска (устанавливаемая на пол)
2.11. гладкость поверхности верхней платформы диска (со стороны пользователя)
2.12. прочность соединения платформ при вращении без нагрузки
2.13. прочность соединения платформ при вращении со средней нагрузкой 45 кг
2.14. прочность соединения платформ при вращении с максимальной нагрузкой 90 кг
2.15. удобство использования пользователем с размером ноги 36 (23 см)
2.16. удобство использования пользователем с размером ноги 44 (28 см)

3. План работы
Стороны договорились следовать следующему плану работы:
1 Подготовка плана тестирования.
2 Согласование тест-плана.
3 Функциональное тестирование.
5 Подготовка отчета.

План и стратегия тестирования
3.1. Функциональное тестирование
Цель функционального тестирования состоит в том, чтобы убедиться, что тренажер работает в соответствии с требованиями и не представляет опасности для пользователя. Функциональное тестирование является наиболее
существенной частью тестирования тренажера, включающее в себя
проверку различных аспектов. Тренажер должен пройти все
запланированные тесты. Только в этом случае можно быть уверенным в его качестве.

3.2. Процедура тестирования
Процедура тестирования предполагает следующие пункты:
- Сообщения об обнаруженных недочетах тренажера.
Необходимо проверить различные аспекты тестируемого тренажера, для
этого требуется выполнение различных типов тестирования.
Основные типы тестирования, которые будут выполнены:
- Функциональное тестирование.
- Юзабилити-тестирование.
- Тестирование безопасности.

3.3. Сообщения об ошибках
Отчеты об ошибках создаются для того, чтобы предоставить команде разработчиков и руководителю проекта исчерпывающую информацию об обнаруженных ошибках.
Каждый баг-репорт содержит следующую информацию о дефекте:
- Название баг-репорта. 
- Краткое описание, представляющее собой краткое описание проблемы.
- Предусловия.
- Шаги для воспроизведения ошибки;
- Ожидаемый результат.
- Фактический результат.

4. Ресурсы
Для тестирования «Диска здоровья» необходимо предоставить помещение со средней комнатной температурой 18-25С и влажностью 35-60%. Пол помещения должен быть твердым и гладким.
Необходимы пользователи весом 45 кг (для тестирования диска со средней весовой нагрузкой) и весом 90 кг (для тестирования с максимальной весовой нагрузкой). Для тестирования размеров диска необходимы пользователи с размером ноги 36 и 44 (24 и 28 см).

5. Критерии качества
Продукт должен работать в соответствии с требованиями и описанием. Продукт не должен содержать критических дефектов, влияющих на безопасность использования.

6. Риски процесса тестирования
Поскольку «Диск здоровья» тестируется при участии человека непосредственно, то существует возможность травмы при тестировании в случае дефектов поверхности (ранения кожи), в случае непрочного соединения платформ — падение пользователя при вращении.
